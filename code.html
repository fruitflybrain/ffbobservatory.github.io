FFBO Installer and Launcher

We provide a repository called FFBO.launcher that can be used to install and launch FFBO in very simple steps.

The installation relies on Docker images hosted on Docker Hub, and the launching is based on docker-compose.

We also provide this Amazon Machine Image on Amazon Web Services for easily launching your own FFBO tools.

Detailed instructions can be found in FFBO.launcher .
